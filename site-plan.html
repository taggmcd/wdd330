Overview

Purpose

The purpose of the website is to offer images that various Mars rovers have taken and make them easily sortable and searchable.

Audience

The target audience are people who are interested in looking at images taken by the various rovers on Mars.

Technical Details

Data Sources

I will be using the following API Mars Photo API

Major Functions

  • Templating: There will be a few of these. They will template the data that will need to repeat such as, Rover info cards, Image cards, Header, and Footer

  • Data Retrieval: This function will get the data from the API

  • Filter Data: This is where the filtering of the data will happen. Such as filtering by which Rover, Day, or Camera a person wants to see images of.

  • Display: There will be a few of these to populate the template functions and actualy insert the ouput into the DOM.
